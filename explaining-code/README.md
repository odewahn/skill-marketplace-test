# Explaining Code Skill Plugin

Makes code explanations more accessible through visual diagrams and real-world analogies.

## Overview

This plugin enhances code explanations by combining multiple teaching techniques: relatable analogies, visual ASCII diagrams, step-by-step walkthroughs, and highlighting common gotchas. Perfect for understanding unfamiliar codebases, code reviews, and learning.

## What This Plugin Does

Every code explanation includes:

1. **Analogy** - Comparison to something from everyday life
2. **Diagram** - ASCII art showing flow, structure, or relationships
3. **Walkthrough** - Step-by-step explanation of what happens
4. **Gotchas** - Common mistakes or misconceptions to watch for

## Installation

```bash
/plugin install explaining-code
```

## Usage Examples

```
How does this function work?
Explain this codebase to me
Can you walk me through what this code does?
Help me understand how this algorithm works
```

## Best For

- Understanding unfamiliar codebases
- Code reviews and documentation
- Teaching and learning programming concepts
- Onboarding new team members
- Breaking down complex algorithms
