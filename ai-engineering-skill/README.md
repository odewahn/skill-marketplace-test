# AI Engineering Skill Plugin

Practical guide for building production ML systems based on Chip Huyen's AI Engineering book.

## Overview

This plugin provides expert guidance on building production-ready AI and ML systems. It covers the full lifecycle from model selection and evaluation to deployment, monitoring, and optimization.

## What This Plugin Covers

- **System Architecture** - Designing and scaling AI applications
- **Dataset Engineering** - Building and maintaining high-quality datasets
- **Model Evaluation** - Defining metrics and evaluation methodologies
- **RAG Systems** - Implementing Retrieval Augmented Generation
- **Finetuning** - Deciding when and how to finetune models
- **Foundation Models** - Selecting and deploying models
- **Inference Optimization** - Optimizing for latency and cost
- **Prompt Engineering** - Designing effective prompts
- **AI Agents** - Building autonomous AI systems

## Installation

```bash
/plugin install ai-engineering-skill
```

## Usage Examples

```
How do I implement a RAG system?

```
